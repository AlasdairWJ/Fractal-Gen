Controls:

  • Left click line: bisect - add new node in the centre
  • Right click line: reflect
  • Left click node: drag to reposition node
  • Right click node: delete node
  • Up arrow: increase number of iterations
  • Down arrow: decrease number of iterations
  • 'g' key: toggle between square and triangular grids
  • Space bar: toggle interface
For the sake of your pc, don't have a lot of nodes with a lot of iterations. Refresh if things get laggy.
